# PBIO Exercise 9
This repository contains a Python program for generating random DNA sequences and saving them in FASTA format.

## Main features:
- random DNA sequence generation
- FASTA file output
- nucleotide percentage statistics
- GC-content calculation
- user name insertion into the generated sequence
- sequence ID and length validation

## Additional features:
- custom nucleotide distribution
- motif search with 1-based positions
- complementary and reverse complementary strands
- in silico transcription to mRNA
